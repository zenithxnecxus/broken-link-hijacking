## 📌 Deskripsi

Broken Link Hijacking Scanner adalah tools untuk mendeteksi potensi kerentanan **Broken Link Hijacking** pada sebuah website. Tools ini akan:

- Crawl semua halaman website
- Mendeteksi link external yang mengarah ke social media
- Memeriksa status akun social media (aktif/tidak)
- Membuat laporan hasil scanning

---

## 🎯 Fitur

| Fitur | Keterangan |
|-------|-------------|
| **Full Website Crawl** | Crawl semua halaman website (bisa diatur max pages) |
| **Social Media Detection** | Instagram, Twitter, Facebook, GitHub, LinkedIn, Telegram, YouTube, TikTok |
| **Status Checker** | Mengecek apakah akun social media masih aktif atau tidak |
| **Random User-Agent** | Menggunakan random User-Agent untuk menghindari block |
| **Save Report** | Menyimpan hasil scan ke file .txt |
| **Color Output** | Tampilan terminal dengan warna warna |
